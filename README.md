# Red-Ant-Assesment

**Family Tree**
A family tree program in c++ implemented using n-ary tree data structure

**Each node has:**
  1. Data of person
  2. Link to first child
  3. Link to next sibiling
  
** Functionalities**
  1. Add new person
  2. Find relationship b/w two persons
  3. Search
  4. Destroy
  5. Display
